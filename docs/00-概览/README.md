## 1. `00-概览` 这一组文档今天到底负责什么

`00-概览` 不是总索引，也不是迁移过程稿存放处。它只保留系统级心智模型：

- 系统整体长什么样
- 关键术语今天怎么读
- 新读者和维护者应该按什么顺序进入文档
- 当 prose、契约与代码冲突时应该信什么

换句话说，概览层回答“先建立什么认知”，不负责重复运行时细节、接口接入细节或专题长链。
